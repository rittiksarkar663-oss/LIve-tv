# Blogger Live TV Grid Theme (RS LIVE TV) This is a custom, fully responsive **Blogger (Blogspot)** XML theme template designed for creating a Live TV streaming website. It features a clean grid layout for TV channels and integrates the **Clappr Player** to play HLS (`.m3u8`) live streams in full-screen mode without leaving the page. --- ## 🚀 Features * **Modern UI:** Built with a dark-themed grid layout (`#0b0c10`) and vibrant neon accents. * **Full-Screen HLS Player:** Integrated with **Clappr Player** for seamless `.m3u8` video streaming. * **Responsive Design:** Auto-adjusting channel grid layout that fits perfectly on Mobile, Tablet, and Desktop screens. * **Dynamic Content:** Embedded standard Blogger widget functionality (`Blog1`) hidden inside the source to maintain clean aesthetics while tracking posts/comments. * **Interactive Hover Effects:** Neon shadows and cards that pop when hovered over. --- ## 🛠️ Tech Stack Used * **Platform:** Blogger (Blogspot) XML Template Engine * **Frontend:** HTML5, CSS3 (CSS Grid & Flexbox) * **Streaming Player:** JavaScript (Clappr Player CDN) --- ## 📖 How to Install on Blogger 1. **Download/Copy the Code:** Copy the full XML code from the repository template file. 2. **Go to Blogger Dashboard:** Navigate to your website backend. 3. **Theme Settings:** Click on **Theme** from the left menu. 4. **Edit HTML:** Click the down arrow next to the "Customize" button and select **Edit HTML**. 5. **Paste & Save:** Delete all existing code, paste this XML code, and click the **Save** icon (floppy disk) in the top right. --- ## 📺 How to Add/Modify TV Channels To add new channels or update existing stream URLs, open the HTML editor and look for the `
